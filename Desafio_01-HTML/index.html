Conceito de HTML

(Voltar)

HTML (Hypertext Markup Language) é o bloco de construção mais básico da web. O HTML é uma linguagem descritiva que especifica a estrutura de uma página web. HTML é uma linguagem de marcação que em conjunto com outras linguagens são usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página da web.

Tags

(Voltar)

As tags são a base da estrutura de qualquer página web. O conhecimento das principais tags HTML é vital para qualquer desenvolvedor front-end.

Abaixo descrevemos tags meramente estrutural e também tags que possuem função estrutural e semântica.

Tags Estruturais

(Voltar)

As tags estrturais ou não semânticas são aquelas que não tem um propósito bem definido, podendo ser utilizadas como uma caixa para posicionar o conteúdo sem adicionar um contexto adicional, pode ser utilizadas para a estilização ou apenas para a organização do código.

Tags Semânticas

(Voltar)

Semântica é um ramo da linguística que estuda o significado das palavras, frases e textos de uma língua. Então podemos afirmar que uma tag pode ter significado semântico e não apenas estrutural.

O HTML não apenas estrutura documentos para a web, mas também descrever o significado do conteúdo presente nesses documentos por meio de tags semânticas. O HTML semântico tem como objetivo descrever o significado do conteúdo presente, tornando-o mais claro tanto para programadores quanto para browsers e outras engines de busca que processam essas informações.

Uma tag semântica é aquela que tem um propósito bem definido, ou que descreve o tipo do seu conteúdo. A tag <h> representa um heading (um cabeçalho) e o número representa a hierarquia desse cabeçalho, ou seja, podemos afirmar que o conteúdo dessa tag <h1> é o cabeçalho principal da página. A tag <h2> representa um subcabeçalho e assim por diante até a tag <h6>, neste caso temos uma hierarquia que devemos observar e analisar dentro de um determinado contexto.

Principais Tags

(Voltar)

Tags <h1> até <h6>

Utilizada para identificar dentro do HTML cabeçalhos e subcabeçalhos hierarquicamente organizados. Possui função semântica ajundando os mecanismos de busca.

Tag <p>

Tag utilizada para definir parágrafos.

Tag <strong>

Utilizado para destacar um texto ou parte dele. Não se caracteriza apenas por deixar o texto em negrito. Este elemento reprenta muito mais importância do que deixar o texto ou parte dele em negrito, possui forte função semântica. No passado era utlizado a tag <b> para deixar o texto em negrito. Atualmente não é mais considerada uma prática recomendada a utilização da tag <b> no HTML5.

Tag <mark>

O elemento <mark> é usado para destacar parte de um texto. Não podemos confundir o elemento <mark> com o elemento <strong>, este é usado para denotar trechos de importância especial, aquele para denotar trechos de relevância especial, funciona como um marca texto.

Tag <small>

Faz um texto ou parte dele ser definido com um tamanho menor. Em HTML5 esta tag é usada para identificar itens de importância secundária, como avisos de direitos autorais, comentários laterias, etc.

Tag <i>

Utilizada para colocar um texto em parte dele em itálico. Trata-se de estética sem função semântica.

Tag <i> VS. tag <em>

Ambos as tags deixam o texto em itálico, porém, a tag <em> indica ênfase textual, aqui o elemento significa algo (semântica), já na tag <i> temos apenas um estilo, taquigrafia tipográfica.

Assim como a tag <b> para negrito, a tag <i> para itálico é uma das tags mais antigas do HTML, No entanto, assim como a tag <b>, o uso da tag <i> não é mais considerado uma prática recomendada no HTML 5. Em vez disso, a tag <em> é recomendada para enfatizar o texto em itálico.

Tag <u>

Originalmente utlizado para deixar um texto sublinhado, foi preterido no HTML 4.01, porém, no HTML 5 foi redefinido para identificar termos com erros ortográficos. Possui uso semântico.

Tag <ol>

É usado para criar uma lista ordenada. Uma lista ordenada é criada aninhando um ou mais elementos <li> entre as tags <ol> de abertura e fechamento.

Tag <ul>

A tag <ul> é usado para definir uma lista não ordenada de itens. Use uma lista não ordenada para conter elementos <li> que não precisam ser apresentados em ordem numérica e podem ser reorganizados sem alterar o significado da lista.

Tag <li>

A tag <li> define um item de lista que faz parte de uma lista ordenada e não ordenada de itens.

Tag <a>

A tag <a> em HTML (ou elemento âncora), com o atributo href cria-se um hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL. O conteúdo dentro de cada <a> precisará indicar o destino do link.

Tag <hr>

Representa uma quebra temática entre elementos de nível de parágrafo (por exemplo , uma mudança da cena de uma história, ou uma mudança de tema com uma seção). Nas versões anteriores do HTML, representava uma linha horizontal. Pode continuar sendo exibida como uma linha horizontal nos navegadores, mas agora está definida em termos semânticos, em vez de termos de apresentação.

Tag <sub>

A <sub> especifica texto embutido que deve ser exibido como subscrito por razões exclusivamente tipográficas.

Tag <sup>

A tag <sup> especifica texto que deve ser exibido como sobrescrito por razões exclusivamente tipográficas.

Tag <blockquote>

A tag <blockquote> indica que o texto incluído é uma longa citação. Normalmente é processado visualmente fica com recuo. Exemplo:

O Aeromodelismo é a arte de planejar e edificar aeromodelos – miniaturas de aeronaves utilizadas com objetivos experimentais, esportivos ou recreativos. Esta atividade envolve a manipulação e a manobra destes objetos de pequena dimensão – aviões, balões, foguetes, entre outros -, construídos pelos aeromodelistas.

Outras Tags

(Voltar)

Vamos analisar outras tags sugeridas.

Tag <font>

Não suportada atualmente em HTML 5. Esta tag foi utilizada em HTML 4 para especificar a face da fonte, o tamanho da fonte e a cor do texto. O que devemos utilizar? A boa prática é sempre utilizar o CSS para as situações acima citada.

Tag <del>

A tag <del> define um texto que foi excluído de um documento. Os navegadores demarcam o texto como tachado. Vejamos um exemplo:

Minha cor favorita é azul vermelho!

No caso acima utilizamos a tag <ins> para definir como texto novo inserido. Ambas as tags possuem significado prório (excluido e apagado), ou seja, pura semântica.

<abbr>

Trata-se de um elemento de abreviação. Isso informa ao usuário o que significa a abreviatura ou sigla. Adicionar um atributo title permite fornecer uma expansão ou definição para a abreviatura ou acrônimo. Exemplo:

CSS é uma linguagem que descreve o estilo de um documento HTML.
